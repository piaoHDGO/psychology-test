# 心理测试网站 - 操作手册

## 一、系统要求

### 1.1 开发环境
- Node.js 18+
- npm 或 yarn
- Git

### 1.2 运行环境
- 现代浏览器（Chrome、Edge、Safari）
- 或微信内置浏览器

## 二、快速开始

### 2.1 本地运行

```bash
# 1. 克隆项目
git clone <repository-url>
cd psychology-test

# 2. 安装前端依赖
cd psychology-test
npm install

# 3. 启动前端
npm run dev
# 访问 http://localhost:3000

# 4. 另开终端，启动后端
cd ../psychology-test-server
npm install
npm start
# 后端运行在 http://localhost:8080
```

### 2.2 快速测试

直接打开构建后的文件：

```
psychology-test/dist/index.html
```

## 三、使用指南

### 3.1 首页功能

1. **查看测试列表**
   - 展示所有可用测试
   - 显示测试名称、简介、价格
   - 点击进入测试

2. **测试分类**
   - MBTI人格测试
   - 心理年龄测试
   - 性格色彩测试

### 3.2 答题流程

1. **开始测试**
   - 阅读题目
   - 选择选项（A/B/C/D）

2. **答题过程**
   - 进度条显示当前进度
   - 自动保存答题进度
   - 可随时退出，下次继续

3. **提交结果**
   - 最后一题点击"查看结果"
   - 计算并展示结果

### 3.3 结果查看

1. **免费结果**
   - 基础类型判断
   - 简要描述

2. **付费解锁**
   - 点击"立即支付解锁"
   - 确认支付
   - 查看完整报告

### 3.4 用户中心

1. **我的测试**
   - 查看测试历史
   - 查看已购买/未购买状态

2. **重新查看**
   - 点击历史记录
   - 查看之前的结果

## 四、管理功能

### 4.1 添加新测试

在 `psychology-test/src/data/quizzes.js` 中添加：

```javascript
{
  code: 'new-test',
  name: '新测试名称',
  description: '测试描述',
  price: 19.9,
  icon: '📝',
  color: '#color',
  category: 'psychology',
  questions: [
    {
      question: '问题内容',
      options: [
        { text: '选项A', score: 1 },
        { text: '选项B', score: 2 }
      ]
    }
  ],
  results: {
    // 结果配置
  }
}
```

### 4.2 修改价格

在题库文件中修改 `price` 字段。

### 4.3 添加结果类型

根据测试类型配置结果：
- MBTI：16种结果类型
- 心理年龄：5个年龄段
- 性格色彩：4种颜色

## 五、支付配置

### 5.1 模拟支付

当前版本使用模拟支付：
- 点击支付后直接解锁
- 用于测试和演示

### 5.2 正式支付

需要配置：
1. 微信商户号
2. 申请开通JSAPI支付
3. 配置支付域名
4. 设置回调地址

## 七、常见问题

### Q1: 答题进度丢失？
A: 检查浏览器localStorage是否正常，可能是隐私模式导致。

### Q2: 结果计算错误？
A: 检查题库配置，确保计分维度正确。

### Q3: 支付无法跳转？
A: 检查微信支付配置，确保域名在白名单中。

### Q4: 后端启动失败？
A: 检查端口是否被占用，尝试更换端口。

## 八、技术支持

如遇到问题，请检查：
1. Node.js版本是否满足要求
2. 依赖是否完整安装
3. 端口是否被占用
4. 数据库连接是否正常
